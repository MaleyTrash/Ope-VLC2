# Ope-VLC2

## Usage
```
play <FILE> - Play file!

pause - Pause playback
resume - Resume playback
exit - Exit...

Volume:
	volume <0-100>
	volume up - Current volume + 5
	volume down - Current volume - 5
	volume doprava - Current volume = 100
Seek:
	seek <DIFFERENCE> - Current time + <DIFFERENCE> Example: seek -10, seek 30
	seek forward - Current time + 5
	seek back - Current time - 5
Playlist:
	playlist - Show/Hide playlist
	playlist add <FILE> - Add file to playlist
	playlist remove <INDEX> - Remove at index
	playlist pop - Remove last
	playlist shift - Remove first
```
